# protosprite-three-demo

An interactive demo application for the [protosprite](https://github.com/brownstein/protosprite) ecosystem, built with React and Three.js.

Try it live at [brownstein.github.io/protosprite](https://brownstein.github.io/protosprite/).

## Features

- Real-time sprite rendering using `protosprite-three`
- Animation playback controls (animation selection, speed, play/pause)
- Visual effects (opacity, outlines, color multiply, fade)
- Layer visibility toggling
- Geometry overlay visualization from `.prsg` polygon data
- Drag-and-drop import of Aseprite exports and ProtoSprite files
- Multi-sprite instancing with bin-packing layout

## Development

```bash
npm start    # Start dev server at http://localhost:3000
npm run build  # Production build to build/
```
