# protosprite-cli

## Command line tooling for protosprite.

```
Usage: protosprite-cli [options]

Utilities for working with protosprite

Options:
  -V, --version               output the version number
  --name [name...]            Provide names for the imported sprites.
  -i, --input [input...]      Process an input file.
  --output [output]           output a ProtoSprite file.
  --external-sheet            output an exernal sprite sheet.
  --preview [preview-output]  output a preview file.
  --json                      output in JSON format
  --debug                     enable debug logging.
  -h, --help                  display help for command
```
